= Manage User Feedback

In the *_Feedbacks_* panel you can review and respond users' emailed *_Feedback_* sent through the form displayed by clicking the footer on the GraphXR *_Projects_* page.

image::/v3/05_01_Feedback_ProjectPage.png[,720,480,role=text-left] 

The feedback form includes user details and the text of the message, as well as the version of GraphXR in use.

The email messages are logged in the admin *_Feedbacks_* panel.

image::/v3/05_02_Feedback_AdminPage.png[,720,480,role=text-left] 

You can:

* Log your response to a user’s email.
+
image::/v3/05_03_Feedback_Marks.png[,720,480,role=text-left] 
+
* Sort messages on any of the fields.
* Search for specific text (i.e. email, subject or content) and selected time period.
* Export *_Selected_* (or *_All_*) user messages as a .CSV file.
