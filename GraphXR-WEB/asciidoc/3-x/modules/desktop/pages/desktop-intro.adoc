= Kineviz Desktop

*Kineviz Desktop* is an application that delivers the full *Kineviz Online* experience running locally on your computer. It provides single-user access to powerful browser-based Kineviz visual analytics, while letting you maintain local control of your data.

[cols="^1,^3"]

* *Memory*: 16 GB or more.
* *Free Space*: 30GB or more
* *Network*: An internet connection is required for initial sign-in and for online features such as Kineviz Agent.

== User Guide

*Kineviz Desktop* is available through a user-managed free subscription (with optional upgrade to paid tiers). It includes connection to a Kineviz-managed cloud server for enabling agentic work. Once downloaded and signed in, you can access the full project management, agent, database connection, file management, data modeling, graph analytic features, and public extensions of *Kineviz Online*. As a single user application, *Kineviz Desktop* does not support multi-user project collaboration.
