= Kineviz Desktop

*Kineviz Desktop* is an application that delivers a full browser-based GraphXR experience running locally on your computer. It provides single-user access to powerful GraphXR visual analytics, while letting you maintain local control of your data.

image::/v3/20_01_01_KD_HomeCards.png[,540,420,role=text-left]

== System Requirements

[cols="^1,^3"]
|===
| *Operating System* | *Supported Devices*
| Windows | Windows 10 / 11 (x64 or ARM64)
| macOS | M3 Mac or better (Intel or Apple Silicon)  
|===

* *Memory*: 16 GB or more.
* *Free Space*: 30GB or more
* *Network*: An internet connection is required for initial sign-in and for online features such as GraphXR Agent.
