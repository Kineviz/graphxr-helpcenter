= Getting Started

*Kineviz* (formerly *GraphXR*) is  visual analytics platform for working with labeled property graphs. 

* *Kineviz Server* is a *multi-user* browser-based application that supports fast, flexible graph modeling and exploration in secure collaborative project workspaces.  

* xref:desktop:desktop-intro.adoc[*Kineviz Desktop*] is a *single-user* download that delivers nearly all the functionality of *Kineviz Server* while running locally on your system.

A *Kineviz* account lets you create and manage multiple projects.

image::/v3/00_01_01_ProjectHome.png[,720,340,role=text-center]
 
== Basic Graph Exploration 

Each *Kineviz* project is a workspace where you can seamlessly import, model and transform graph data, explore and analyze graph patterns, save your work, and share your insights. *Kineviz* projects also include a built-in AI *_Agent_* for optional graph modeling and exploration through AI chat.

== Iterative Visual Analytics with Kineviz 
  
Since exploring a graph quickly sparks new questions, graph workflows are highly iterative. As an exploration proceeds, you will most likely need to simplify or transform the graph, edit or add to the data already in the graph, focus on specific subgraphs, and save and communicate key views of the graph as you go.
