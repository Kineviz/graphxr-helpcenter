= GraphXR Explorer for BigQuery

GraphXR Explorer provides a one-stop connection from the Google Cloud Marketplace to a BigQuery property graph. 

== Features

* Single Project
* Single User
* Supports up to 50,000 nodes and 50,000 edges
* Streamlined visualization for BigQuery.
* User-managed *UI Configuration* for additional GraphXR features such as Transforms, Map, Algorithms, and Extensions.

=== Before you begin…

Before you can use GraphXR Explorer, you need:

* A Google Cloud BigQuery account.
* BigQuery data sources and a BigQuery property graph database modeled from them.
+
NOTE: You can start with BigQuery demo data available for free trial through your Google Cloud Platform account.
