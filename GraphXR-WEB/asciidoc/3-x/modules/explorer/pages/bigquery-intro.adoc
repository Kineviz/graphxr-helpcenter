= GraphXR Explorer for BigQuery

*GraphXR Explorer for BigQuery is under development. Please link:https://kineviz.com/contact[contact us] for more information.*

== Features

* Single Project
* Single User
* Supports up to 50,000 nodes and 50,000 edges
* Streamlined visualization for BigQuery.
* User-managed *UI Configuration* for additional GraphXR features such as Transforms, Map, Algorithms, and Extensions.
