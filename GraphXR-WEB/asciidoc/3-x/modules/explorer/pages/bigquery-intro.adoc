= Kineviz Explorer for BigQuery

Kineviz *Explorer* provides a one-stop connection to Google BigQuery datasets from the 
https://console.cloud.google.com/marketplace/product/kineviz-public/graphxr-explorer-for-bigquery?q=search&referrer=search&invt=Abtgig&project=kineviz-public[*Google Cloud Platform (GCP) Marketplace*].

NOTE: *Kineviz Explorer for integrated BigQuery / BigQuery Graph is under development. Please link:https://kineviz.com/contact[contact us] for more information.*

== Features

* Single Project
* Single User
* Supports up to 50,000 nodes and 50,000 edges
* Streamlined visualization for BigQuery.
* User-managed *UI Configuration* for additional *Kineviz* features such as Transforms, Map, Algorithms, and Extensions.
