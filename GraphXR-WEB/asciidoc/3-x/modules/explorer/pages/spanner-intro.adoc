= Kineviz Explorer for Spanner

Kineviz *Explorer* provides a one-stop connection to a property graph on Google Spanner from the link:https://console.cloud.google.com/marketplace/product/kineviz-public/graphxr-explorer-spanner?q=search&invt=Abtgig&project=kineviz-public[*Google Cloud Platform (GCP) Marketplace*]. 

xref:attachment$quickstart-card-spanner.pdf[*Kineviz Explorer for Spanner: Quick Start Card*]

== Features

* Single Project
* Single User
* Supports up to 50,000 nodes and 50,000 edges
* Streamlined visualization for Spanner Graph.
* User-managed *UI Configuration* for additional *Kineviz* features such as Transforms, Map, Algorithms, and Extensions.
