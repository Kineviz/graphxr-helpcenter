= GraphXR Explorer for Spanner

GraphXR Explorer provides a one-stop connection to a property graph on Google Spanner from the Google Cloud Marketplace. 

xref:attachment$quick-start-card.pdf[*Quick Start Card*] 

link:https://console.cloud.google.com/marketplace/product/kineviz-public/graphxr-explorer-spanner?q=search&invt=Abtgig&project=kineviz-public[Deploy on Google Cloud Marketplace]

== Features

* Single Project
* Single User
* Supports up to 50,000 nodes and 50,000 edges
* Streamlined visualization for Spanner Graph.
* User-managed *UI Configuration* for additional GraphXR features such as Transforms, Map, Algorithms, and Extensions.

=== Before you begin…

Before you can use GraphXR Explorer, you need:

* A Google Cloud Spanner account.
* A Spanner database and Spanner Graph property graph database based on it.
+
NOTE: You can start with Spanner Graph demo data available for free trial through your Google Cloud Platform account.
