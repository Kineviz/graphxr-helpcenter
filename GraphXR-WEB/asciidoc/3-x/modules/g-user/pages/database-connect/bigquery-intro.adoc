== Connection Options

* GraphXR *_Explorer_* provides a one-stop connection from the Google Cloud Marketplace to to a single project and a limited amount of data held in a BigQuery data source. 

* GraphXR *_Enterprise_* provides full GraphXR collaborative multi-user access and unlimited projects.
