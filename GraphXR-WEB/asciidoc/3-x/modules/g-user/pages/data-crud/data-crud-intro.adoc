= Create and Edit Data

Graph data can be easily created, edited, and deleted within *Kineviz*. At any time you can:

* xref:data-crud/data-add.adoc[*_Add nodes and edges_*] and their properties.
* xref:data-crud/data-edit.adoc[*_Add and edit labels_*] of categories, relationships, and properties.
* xref:data-crud/tag-nodes.adoc[*_Tag_*] selected data to label and save any selection of nodes. 
* xref:data-crud/data-delete.adoc[*_Delete_*] nodes, edges, and properties, clean out unused category or relationship metadata, and empty or delete the internal graph database.
