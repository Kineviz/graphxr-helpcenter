= Kineviz User Guide

https://www.kineviz.com/graphxr[*Kineviz*] is a browser-based visual analytics platform for exploring labeled property graphs. A *Kineviz* project is a virtual 2D or 3D workspace designed to accelerate exploration and foster collaborative investigation and discovery. 

*Kineviz Desktop* enables single-user offline use. As a standalone application installed it on your system, it delivers graph visualization and analytics that's nearly identical to the *Kineviz* online experience.    

== The Kineviz Platform 

image::/v3/01_00_01_UserGuideIntro.png[,720,380,role=text-center] 

In a *Kineviz* project you can:

* Seamlessly import graph data from:

** A graph database via GQL, Cypher, or visual no-code graph query. Supported databases include Google SpannerGraph, Memgraph, Puppygraph, Neo4j, and more.
** A relational database via SQL or Azure DB via Gremlin.
** CSV and JSON via drag and drop. 

* Explore graph data efficiently.

* Model, transform, and simplify complex graphs iteratively through data mapping and powerful data transforms, and save your work to a project's views and its internal datastore. 

* Discover graph patterns through widely used graph algorithms for *Path Finding*, *Centrality*, and *Community Detection*.

* Create compelling graph visualizations through flexible layouts and geospatial mapping, and through project templates for saving your color, icon, and captioning choices.
 
* Save, share, and export a *Kineviz* project, its data views and internal *KoreDB* datastore.

* Securely collaborate via managed users, permissions, and data sources.
+
NOTE: *Kineviz* project sharing is not available in the single-user *Kineviz Desktop* application.

== Supported Environments

* WINDOWS, MAC OSX, AND LINUX

* CLOUD, PRIVATE CLOUD, AND ON-PREMISES DATA HOSTING.

* The *Kineviz* client runs best in Google Chrome. Compatibility with other browsers may vary.
