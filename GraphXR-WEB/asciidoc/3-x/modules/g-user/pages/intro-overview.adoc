= Kineviz User Guide

*Kineviz Server* is a browser-based visual analytics platform for exploring the  connections in data modeled as labeled property graphs. Its multi-user virtual 2D or 3D project workspaces are designed to accelerate exploration and foster collaborative investigation. 

image::/v3/01_00_01_UserGuideIntro.png[,720,380,role=text-center] 

NOTE: xref:desktop:desktop-intro.adoc[*Kineviz Desktop*] is available as a single-user application that you download to your system. It does not support project sharing; otherwise its features and UI elements are nearly identical to *Kineviz Server*.     

== The Kineviz Platform 

In the *Kineviz* platform, you create and manage persistent projects that can be connected to graph data sources and shared with other users. You can:

* *Import* data in various ways:

** *Query a connected graph database* via visual no-code graph query, Cypher, or GQL. Supported databases include Google Spanner Graph and BigQuery Graph, Neo4j, Memgraph, Puppygraph, UltipaGraph, OracleGraph, GalaxyBase, NebulaGraph, and more.
** *Query a connected relational database* via SQL or Gremlin.
** Open (or drag and drop) JSON files and CSV and Excel files or archives. 

* *Model, transform, and simplify graphs* iteratively through data mapping and powerful data transforms, and save your work to a project's views and internal *KoreDB* data store. 

* *Discover graph patterns* through widely used graph algorithms for *Path Finding*, *Centrality*, and *Community Detection*.

* *Create focused graph visualization* through color, captioning, size scale, selection and filtering, flexible layout, and geospatial mapping. 

* *Save a project template* that specifies color, icon, and captioning choices.
 
* *Save, share, and export* a *Kineviz* project, its data views and internal *KoreDB* datastore.

* *Securely collaborate* via managed users, permissions, and data sources.
 
== Supported Environments

* WINDOWS, MAC OSX, AND LINUX

* CLOUD, PRIVATE CLOUD, AND ON-PREMISES DATA HOSTING.

* The *Kineviz* client runs best in Google Chrome. Compatibility with other browsers may vary.
