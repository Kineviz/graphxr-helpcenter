= Select Graph Data

The options  available to select subsets of the data on the canvas include: 

* Select visible nodes and edges on the canvas

** xref:./ref-shortcut-keys.adoc[Mouse and/or keyboard controls] to select and deselect visible nodes and edges, and add to or subtract from your selection.

** Toolbar one-click *_Select Visible_* (*_Ctrl + A_*) and *_Inverse_* (*_Ctrl_* *+* *_R_*) icons. 

* Select by category, relationship, or property
 
** xref:./select.adoc[*_Legend_*] lists of data by category, property, tag, or relationship. You can select all nodes of one or more categories or tags, edges of one or more relationships, or nodes or edges that include one or more listed values of a property.

** xref:./navigate.adoc[*_Search_*] property values in the graph on the canvas, and select search results.

** Search and select from xref:./table-explore.adoc[*_Tables_*] of data on the project canvas by category and relationship.

* xref:./filter.adoc[*_Filter_*] and select data on the canvas by numerical, date, or categorical property value. 

* Select based on xref:./select-paths.adoc[*_Connections and Paths_*]

** Right-click *_Select_* to select and deselect all nodes, select nodes or edges connected in a variety of specific ways, and to add or subtract a node or edge from a current selection.

** Toolbar *_Collect Nodes_*] and *_Explode Collections_* icons to collect nodes with few (or no) connections to selected nodes, and place them in temporary collection nodes.

** Right-click *_Find Path_* to find and select the connected nodes and edges between any two selected nodes. 

** Toolbar *_Trace Neighbor_* and *_Select Visible_* icons, to highlight and select paths a specific number of hops away from a selection of nodes.  

With data selected, you can:

* xref:./hide-show.adoc[*_Hide_*] selected data temporarily, with toolbar *_Hide Selection_* and *_Un-Hide Selection_* , or right-click *_Actions > Hide_* and *_Actions > Show Hidden_* menu items. 

* xref:data-crud/tag-nodes.adoc[*_Tag_*] any selection of nodes, label it, and list it in the Legend as a *_Tag_*.
