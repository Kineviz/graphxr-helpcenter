= Navigate the Graph

To navigate the graph on the project canvas:

* Use mouse and/or keyboard navigation controls.   
 
* Reposition your view of the graph with one-click *_Fly Out_* and *_Center To_* icons on the toolbar.
* Search on property values, and center the graph on matching nodes or edges.
* Search and center on one or more nodes or edges using a xref:./table-explore.adoc[*_Table_*].

== Mouse and Keyboard navigation

Navigation controls let you manually pan, rotate, and zoom in and out of the entire graph,  and move all or part of the data on the canvas. Rotation and zoom functions center on selected nodes, if any.

== Fly Out and Center To

The *_Fly Out_* and *_Center To_* toolbar icons reposition your view of the graph.

image::/v3/04_01_02_FlyOutCrop.png[,420,240,role=text-left]

* Click *_Fly Out_* to display all the graph data centered in your browser window. Or, click the *_Reset_* circle in the GUI navigation controls.
* Click *_Center To_* to zoom in to a view of the data centered on a selected node or nodes.
 
== Navigate using a keyword search

The search bar at the top left provides full-text keyword search on properties of nodes on the project canvas.  

Nodes with matching property values are highlighted as you enter search terms. You can then click items on the list to select and center the graph on the selected data. 

*To navigate using keyword search of property values:*

. If your project is connected to a database, click the icon in the search bar and choose  to search the data already on the canvas (rather than query the database).

. Enter a full or partial search term. For example, to find a _Character_ node with the _name_ property value of _Grey Wind_, you can simply enter "`win`".
+
image::/v3/04_01_05_KeywordGraph.png[,520,380,role=text-left]

+
As you enter the term, nodes with property values matching the search term are selected. A list of results by category and node appears below the search bar. The example search returned two _Character_ nodes, one of which has the _name_ property value we want.

+
NOTE: In a graph search partial text is entered without asterisks. Any space between search terms is recognized as *OR* logic. *AND* logic is not recognized. 

. For more details, you can mouse over a list item for an information window.
. Click anywhere on an item to select the node and center it in the project space.   
+
image::/v3/04_01_06_KeywordFlyTo.png[,720,480,role=text-left]

. Click the *X* in the search bar to clear a search.

== Navigate using a Table

In a table of the data for any category or relationship, you can click a table row to select a node or edge and center it on the canvas. 


*To navigate to a node or edge from a table:*

. Select data in the graph using any method, or double-click in an empty area to de-select all data. 
. Open the *_Table_* panel, click the *_Category_* or  *_Relationship_* tab, and a category or relationship label to display a table of the nodes or edges. 
. To limit a table to nodes or edges with property values of interest, use the  *_Table_* panel's *_Search_* bar.   
. Click a table row for a single node (or edge).
+
The graph centers on the node (or edge), highlighting but not selecting it. You can select the node or edge on the canvas with *_left mouse click_*, or if the search has returned the one item of interest, click the *_Select All_* button below the table to select it.
