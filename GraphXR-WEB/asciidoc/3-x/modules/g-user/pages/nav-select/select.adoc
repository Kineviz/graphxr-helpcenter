= Select Data on the Canvas

Selecting nodes and edges on the canvas can be done using:

* Mouse and keyboard selection controls
* Toolbar *_Select Visible_*  and *_Inverse_* icons. 
* One-click selection in the Legend by category, property, tag, relationship, or relationship property.
* xref:./navigate.adoc[*_Search_*] of property values in the graph, and selection of search results.
* xref:./table-explore.adoc[*_Tables_*] of data by category and relationship, to select, center on, or edit data on the project canvas.  

== Select data with mouse and keyboard controls

Use standard mouse and/or keyboard xref:./ref-shortcut-keys.adoc[_Selection Shortcuts_] to select visible data on the project canvas. You can select one or more nodes and/or edges and add to or subtract from your selection. 

Once you have a selection, you can:

* Use *_left mouse click_* *drag* to manually move a selection to another part of the canvas and pin the selection to that location. 

* Click *_Inverse_* on the toolbar or the right-click *_Select_* menu to deselect currently selected nodes and edges, and select all the other visible data. The badge on the toolbar icon shows the number of currently selected nodes.
+
image::/v3/04_02_05_Inverse.png[,720,120,role=text-left]
+

+
NOTE: *_Inverse_* is often used along with xref:./hide-show[*_Hide / Unhide Selection_*] to simplify a graph by temporarily removing and then restoring some of its data.
+

* xref:data-crud/tag-nodes.adoc[*_Tag_*] and label any selection, which lists it in the Legend's *_Tag_* section.

== Select in the Legend

The Legend displays scrollable lists of nodes or edges currently visible in the graph by

* *_Category_*
* *_Property_* values of Categories
* *_Tag_*
* *_Relationship_*
* *_RelProperty_* property values of Relationships
 
Click on a Legend section to display the lists of the nodes or edges. By default, the legend displays the category list. You can click the Relationship section to display the list of relationships below the categories.

image::/v3/04_02_00_LegendSelect.png[,520,320,role=text-left]

List items appear in bubbles that include the label, the total number of nodes or edges of that type, and the number currently selected. The list is displayed in descending order of the amount of data: items with the most nodes (or edges) appear at the top of a list. You can:

* Click a list bubble to choose all its nodes (or edges). 
* *_Ctrl+ left click_* to select more than one list item.

TIP: Use the mouse wheel or scroll gesture to scroll through long lists.

=== Select by Category, Tag, or Relationship

. In the Legend, click Category, Tag, or Relationship
. Click a list item to select: 
+
All the nodes of a Category or Tag, or
+
All the edges of a Relationship
+

=== Select by Property value

. In the Legend, click *_Property_* or *_RelProperty_*
. Use the dropdown menu to choose the Category (or Relationship) and Property Name.

+

TIP: You can see a list of all property names, but since they do not have to be unique across the database, it's best to select the property name for a specific Category.

== Select using a Table 

*To search and select from a table:*

. Select data in the graph using any method, or double-click in an empty area to de-select all data. 
. Open the *_Table_* panel, click the *_Category_* or  *_Relationship_* tab, and a category or relationship label to display a table of the nodes (or edges). 
. In the table's *_Search_* bar, enter a search term to show the rows for items with any property value matching the search term. 

. Click *_Select All_* to select all the data that match your search.
+
image::/v3/04_02_04_TableSearchSelect.png[,520,380,role=text-left]
+ 

+
TIP: A table search followed by *_Select All _* lets you select multiple nodes (or edges) with similar property values, for example _Location_ nodes with the same _postalCode_.
